Este código refere-se a implementação das funções do controlador central (CC) utilizada nas validações do modelo dinâmico teórico e nas estratégias de controle 
propostas na tese de doutorado de Lucas Silvestre Chaves entitulada por "Modelo Dinâmico de uma Microrrede Avançada de Energia Elétrica com Controle
Centralizado sob Latência de Comunicação". O código foi desenvolvido na plataforma Node-RED utilizando Java-Script para ser embarcado no Raspberry-Pi 5.
Mais detalhes da implementação, das funcionalidades do CC e das validações estão disponíveis na tese.

This code refers to the implementation of the central controller (CC) functions used in the validation of the theoretical dynamic model and the control strategies proposed in the PhD thesis by Lucas Silvestre Chaves entitled "Dynamic Model of an Advanced Electrical Energy Microgrid with Centralized Control under Communication Latency". The code was developed on the Node-RED platform using JavaScript to be deployed on a Raspberry Pi 5. More details about the implementation, CC functionalities, and validation procedures are available in the thesis.
